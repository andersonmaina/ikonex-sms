# Ikonex Academy

A school management system for managing students, subjects, assessments, analysis and generating report cards.

## Stack

- **Frontend**: React + TypeScript + Vite (deployed on Vercel)
- **Backend**: Node.js + Express + TypeScript (deployed on Railway)
- **Database**: Supabase (PostgreSQL)
- **Version Control**: Git / GitHub

## Local Development

### Frontend
```bash
cd frontend
pnpm install # or npm install

pnpm dev     # or npm run dev
```

### Backend
```bash
cd backend
pnpm install # or npm install
pnpm dev     # or npm run dev
```

## Environment Variables

Copy `.env.example` files and fill in your values
